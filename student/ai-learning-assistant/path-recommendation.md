# AI学习助手

**个人助手设置 ：**进入智慧学伴界面，教师和学生可以查看并修改个人助手的配置。在配置页面，可以根据自己的需求和偏好，对个人助手的功能、界面等进行定制，使其更符合个人使用习惯。此外，页面下方还展示了一些智能体，教师和学生可以了解它们的功能和用途，以便在学习过程中使用这些智能体辅助教学。

## 学习路径推荐

提供通过分析学生知识掌握情况，运用个性化推荐算法，精准推送定制化学习内容，涵盖课程学习顺序、拓展阅读资料、实践训练建议等
