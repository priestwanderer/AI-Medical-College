# 操作流程
1.点击 `“修改配置”`按钮，在弹窗的提示词输入框中编辑个性化身份 / 需求描述（如 “我是一名妇产科学学生”）。  
2.编辑完成后点击 `“保存”`，完成提示词配置。  
3.点击 `“点击使用”` 按钮，进入助手交互页面。  
4.在助手交互页面点击 `“配置知识范围”`，选择专业知识库分类（如 ESMO、ESGO、BGCS 等医学学会相关资源），可通过勾选指定知识来源；同时提供大模型选择功能，适配不同的回答需求。  
5.在输入框输入问题，即可发起咨询。
